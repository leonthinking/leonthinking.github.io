 # Leon Zhang's Personal Website

这是我的个人主页项目，基于 GitHub Pages 构建。访问地址：[leonthinking.github.io](https://leonthinking.github.io)

## 🚀 特性

- 响应式设计，支持移动端访问
- 优化的 SEO 配置
- 社交媒体分享优化
- 支持中英文双语页面
- 支持浅色 / 深色主题切换，并记住用户偏好
- 作品、文章、生活剪影与联系方式模块化展示
- 页面加载优化

## 🛠️ 技术栈

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages

## 📦 项目结构

```
leonthinking.github.io/
├── index.html          # 默认入口，跳转到中文页面
├── cn.html             # 中文主页
├── en.html             # 英文主页
├── favicon-32x32.png   # 网站图标（32x32）
├── favicon-16x16.png   # 网站图标（16x16）
├── apple-touch-icon.png # iOS 设备图标
├── robots.txt          # 搜索引擎爬虫配置
├── sitemap.xml         # 中文与默认入口网站地图
├── sitemap-en.xml      # 英文页面网站地图
└── README.md           # 项目说明
```
